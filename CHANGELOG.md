# Change Log

All notable changes to the **BookNook** extension are documented in this file.

## [0.1.0] - 2026-07-07

### Added

- Bookshelf view with cover grid, reading progress, import and remove
- EPUB and TXT reading in the sidebar or main editor area
- Reading progress memory (scroll position, chapter, CFI for EPUB)
- Chapter navigation, table of contents, and full-text search
- Theme-aware reader UI that follows VS Code / Cursor color themes
- Settings for font size, line height, and reader location

### Notes

- First public release on the Visual Studio Marketplace
